# Fundamentos de Python 1 - Cisco Networking Academy

Este repositorio documenta mi proceso de aprendizaje y consolidación de conocimientos durante el curso Fundamentos de Python 1 de Cisco Networking Academy.

## Método de Aprendizaje
El propósito de este repositorio no es únicamente almacenar el contenido del curso, sino servir como registro de mi práctica. Para asegurar la comprensión de los conceptos y garantizar la superación de los exámenes de cada módulo, realicé una serie de ejercicios prácticos diseñados por mí mismo antes de realizar a las pruebas oficiales.

Este método me permitió:
- Entender la sintaxis y la lógica de programación antes de cada evaluación.
- Crear una biblioteca de scripts funcionales y reutilizables.
- Aplicar los fundamentos básicos de Python de forma práctica.

## Estructura del Repositorio
El contenido está organizado por bloques alineados con los módulos del curso:

* **01_introduccion_python:** Configuración del entorno y primeros programas básicos.
* **02_tipos_datos_y_basicos:** Variables, tipos de datos y operadores fundamentales.
* **03_control_de_flujo:** Lógica de condiciones y bucles.
* **04_funciones_y_datos:** Funciones modulares, estructuras de datos y manejo de excepciones.
* **proyectoFinal:** Aplicación integrando todos los conocimientos adquiridos.

## Tecnologías Utilizadas
- **Lenguaje:** Python 3
- **Entorno de desarrollo:** PyCharm
